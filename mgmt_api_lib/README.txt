A java library that simplifies the usage of the Check Point management APIs.

Folders structure:
 (*) jar - Jar file of the library. Add it to the classpath to use it in your project
 (*) src - Sources of the library
 (*) lib - The library dependencies
 (*) mgmt_api_docs - The library JavaDoc pages

In order to use this library in your project, add its JAR file with dependencies from the lib folder to the classpath
