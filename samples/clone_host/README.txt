This example demonstrates cloning a given host object, so that the cloned host will appear in the same groups
as the original host.

Folders structure:
 (*) src - Sources of the example.
 (*) lib - The example dependencies.

In order to run this example:
Insert in the main function yours management server credentials : server IP, user name, password.
Also insert the name of the original host and name and IP of the new host.

To read the example code, load the sources to an IDE and add the dependencies from the lib folder to project's
classpath.
